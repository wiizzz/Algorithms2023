# PA3

### DIRECTORY:

    .
    ├── bin                   # executable binary
    ├── doc                   # reports
    ├── inputs                # input data
    ├── outputs               # output result
    ├── src                   # source C++ codes
    └── utilities             # checker


### HOW TO CIMPILE:
	make all

### HOW TO RUN:

	cd bin/
	./bin/cb <input_file_name> <output_file_name>
